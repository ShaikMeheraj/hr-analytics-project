# hr-analytics-project
Excel HR Analytics Dashboard for Attrition Analysis, KPI Reporting and Workforce Insights

## Project Overview
Built an interactive HR Analytics dashboard using Excel to analyze employee attrition, workforce trends, and salary patterns, supporting data-driven decision making.

## Key Analysis Performed
- Attrition Distribution Analysis
- Department-wise Attrition Trends
- Salary Analysis by Job Role
- Job Role Attrition Analysis
- Attrition by Years at Company

## Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Dashboarding
- Data Visualization

## Skills Demonstrated
- Exploratory Data Analysis (EDA)
- HR Analytics
- KPI Reporting
- Dashboard Development
- Business Insight Generation

## Key Outcomes
- Identified major attrition trends across departments and job roles
- Analyzed compensation patterns by employee role
- Generated insights to support retention and workforce planning strategies
